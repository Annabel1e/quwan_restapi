 
**简要描述：** 

- 搜索

**请求URL：** 
- ` http://xx.com/quwan/search `
  
**请求方式：**
- POST 

**参数：** 
```
{
    key: 关键字 | 字符串 | 必填
    type:  类型 | 整形 | 否 [1景点,2目的地，3路线,4节日，5酒店,6餐厅]
} 

```
