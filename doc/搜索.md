 
**简要描述：** 

- 搜索

**请求URL：** 
- ` http://xx.com/quwan/search `
  
**请求方式：**
- POST 

**参数：** 
```
{
    key: 关键字 | 字符串 | 必填
    filter:  搜索对象 | 整形 | 必填 [0全部(默认),1景点,2目的地，3路线,4节日，5酒店,6餐厅]
    sortby:  搜索排序 | 字符串 | 否 [key相关优先(默认), distance距离优先, score评分优先
    lon:  用户经度 | 字符串 | 否 [当搜索排序为距离优先时必传]
    lat:  用户纬 | 字符串 | 否 [当搜索排序为距离优先时必传]
    cid:  搜索分类id | int | 否 
} 

```
