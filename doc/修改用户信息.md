 
**简要描述：** 

- 修改用户信息

**请求URL：** 
- ` https://restapi.qu666.cn/quwan/edit_user_info `
  
**请求方式：**
- post

**参数：** 
```
{
    user_nickname:  昵称 | 字符串 | 必填
    user_avatar:  头像 | 字符串 | 必填
    user_mobile:  手机号 | 字符串 | 必填
    user_sex:  性别[0女,1男] | 整形 | 必填
} 

```




 **成功返回**
```
{"msg":"修改成功"}
```

 **失败返回** 

```
{
    "code": "122850005",
    "message": "参数不合法,请检查"
}
```
