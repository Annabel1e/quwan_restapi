 
**简要描述：** 

- 获取用户订单统计信息

**请求URL：** 
- ` http://xx.com/quwan/order_count `
  
**请求方式：**
- GET 

**参数：** 
 无




 **成功返回**
```
{
    "pay_no": 0, #待支付
    "pay_ok": 0, #可使用
    "score_no": 0, #待评价
    "order_ok": 0, #已完成
    "order_refund": 0 #已取消
}
```

 **失败返回** 

```

{
    "code": "122850000", 
    "message": "未登录或没有权限"
}
```
