 
**简要描述：** 

- 评价列表

**请求URL：** 
- ` http://xx.com/quwan/score `
  
**请求方式：**
- get

**参数：** 
```
{
    score_type:  评价类型[1景点,2节日，3酒店,4餐厅] | 整形 | 必填
    limit:  每页显示数 | 整形 | 必填
    offset:  页码 | 整形 | 必填
} 

```
