 
**简要描述：** 

- 发送短信验证码

**请求URL：** 
- ` https://restapi.qu666.cn/quwan/send_sms `
  
**请求方式：**
- post

**参数：** 
```
{
    user_mobile:  手机号 | 字符串 | 必填
} 

```




 **成功返回**
```
{"msg":"发送成功"}
```

 **失败返回** 

```
{
    "code": "122850005",
    "message": "参数不合法,请检查"
}

```

 **备注** 
```
发送成功后的验证码失效时间是  1分钟
```
