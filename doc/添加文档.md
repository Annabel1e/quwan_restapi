 
**简要描述：** 

- 添加文档到索引

**请求URL：** 
- ` http://xx.com/quwan/add_index `
  
**请求方式：**
- post

**参数：** 
```
{
    id:  主键id | 整形 | 必填
    type:  类型 | 整形 | 必填 [1景点,2目的地，3路线,4节日，5酒店,6餐厅]
    name:  名称 | 字符串 | 必填
    img:  图片 | 字符串 | 必填
    intro:  介绍 | 字符串 | 必填
    lon:  经度 | 浮点 | 必填
    lat:  纬度 | 浮点 | 必填
    sort:  排序 | 整形 | 必填
    created_at:  创建时间 | 整形 | 必填
    address:  地址 | 字符串 | 否
    phone:  电话 | 字符串 | 否
    price:  价格 | 浮点 | 否
    score:  评分数 | 浮点 | 否
    evaluation:  评价数 | 整形 | 否
    open_time:  开放时间 | 字符串 | 否
    suggest:  建议 | 字符串 | 否
    sales_num:  销售数 | 整形 | 否
    score_num:  评论数 | 整形 | 否
} 

```




 **成功返回**
```
{"msg":"操作成功"}
```

 **失败返回** 

```
{
    "code": "122850015",
    "message": "缺少类型"
}

```
